# Survival Commands

Use this page for common Survival commands.

## Syntax

- `[]` = required argument
- `()` = optional argument

## Homes

By default, players have `2` homes.

You can unlock more homes with ranks, and your placed bed location also works as a teleport point.

| Command | Description |
| --- | --- |
| `home (homeName)` | Teleport to a home location. |
| `homes` | Opens a clickable menu with your homes. |
| `removehome (homeName)` | Removes a home. |

## Economy and Utility

| Command | Description |
| --- | --- |
| `balance` | Check your money balance. |
| `pay [playerName] [amount]` | Send E-Maralds to another player. |
| `playtime` | Show your total playtime. |
| `spawn` | Teleport back to spawn. |
| `shakeitoff` | Dismount any entity riding you. |

## Chat Rooms

| Command | Description |
| --- | --- |
| `chat create [chatName] (-private)` | Creates a chat room. Add `-private` to make it invite-only. |
| `chat join [chatName]` | Joins a chat room. |
| `chat leave [chatName]` | Leaves a chat room. |
| `chat list` | Lists available chat rooms. |
| `chat listrooms` | Shows your current/known chat rooms. |
| `chat invite [chatName] [playerName]` | Invites a player to a room. |
| `chat kick [chatName] [playerName]` | Removes a player from a room. |

These commands are used to create and manage player chat rooms.

## Mail

| Command | Description |
| --- | --- |
| `mail send [playerName] [message]` | Sends mail to a player. |
| `mail read` | Reads your inbox/mail entries. |
| `mail clear` | Clears your mail entries. |

`sendtemp` is admin-only and is not available to regular players.


## Teleport Requests

| Command | Description |
| --- | --- |
| `tpa [playerName]` | Ask to teleport to another player. |
| `tpaccept (playerName)` | Accept a teleport request. |
| `tpahere [playerName]` | Ask a player to teleport to your location. |
| `tpdeny (playerName)` | Deny a teleport request. |

## Warps

| Command | Description |
| --- | --- |
| `warp (warpName)` | Teleport to a warp. Without a name, opens the warp menu. |

For player-created warps and management commands, see [Player Warps](/servers/survival/player-warps).
