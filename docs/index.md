## How to Join

### Java Edition (PC)

1. Open Minecraft Java Edition.
2. Go to `Multiplayer` -> `Add Server`.
3. Enter the server address: `play.worstserverever.com`
4. Save and join.

### Bedrock Edition (PC and Mobile)

1. Open Minecraft Bedrock Edition.
2. Go to `Play` -> `Servers` -> `Add Server`.
3. Enter the server address: `play.worstserverever.com`
4. Use port `19132`.
5. Save and join.
